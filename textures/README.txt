Put your planet texture files here.

Expected filenames (these match Solar System Scope's default names):

  2k_sun.jpg
  2k_mercury.jpg
  2k_venus_atmosphere.jpg
  2k_earth_daymap.jpg
  2k_moon.jpg
  2k_mars.jpg
  2k_jupiter.jpg
  2k_saturn.jpg
  2k_saturn_ring_alpha.png   (MUST be PNG with alpha channel)
  2k_uranus.jpg
  2k_neptune.jpg

If your files are named differently, either rename them or edit the
texturePath fields in src/SolarSystem.cpp.

Missing textures will render as solid magenta; the program will still run.

Free textures: https://www.solarsystemscope.com/textures/
